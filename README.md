# UserManagementAPI

This project is a simple ASP.NET Core Web API developed for managing users.

## Features

- CRUD API endpoints
- Input validation
- Logging middleware
- Error handling middleware
- Authentication middleware

## Endpoints

GET /users  
GET /users/{id}  
POST /users  
PUT /users/{id}  
DELETE /users/{id}

## Authentication

All requests require header:

Authorization: secrettoken
